
__This is a core component of [Wolfram JS Frontend](https://github.com/JerryI/wolfram-js-frontend) project__

## Features
- `70-80%` coverage of Wolfram Language `Graphics`, `Image`
- Dynamics support
- Fully functional SVG graphics with panning, zooming, event bindings
- `GraphicsComplex` WebGL acceleration
- Automatic triangulation of complex polygons
